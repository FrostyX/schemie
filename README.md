# schemie
Web-based Scheme IDE


## Features
- Editor
    - [ ] Syntax highlighting
    - [ ] Bracket matching (each pair of brackets toned into own color)
    - [ ] Lisp-style indentation
	- [ ] Buttons: Save, Run
- Command line
    - [ ] Going through commands history via arrow keys

## Running

	python run.py
